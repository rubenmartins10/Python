- **count()** : este método é usado para determinar quantas vezes um item aparece numa tupla
  - Utilizamos da seguinte maneira :
    - programacao_linguagens.**count(*Palavra*)**
  - Se nenhum argumento for passado para a função, então recebemos um **TypeError**

- **index()** : este método é usado para encontrar o índice onde um determinado item está presente numa tupla.
  - programacao_linguagens.**index(*Palavra*)**
    - Se o item especificado não for encontrado, recebemos um **ValueError**
  - programação_linguagens.index('python', 3)
    - Começa a por procurar a palavra python na tupla a partir do índice 3
  - programação_linguagens.index('python', 2, 5)
    - Começa a procurar a palavra python na tupla a partir do índice 2 e a acabar no 5, não inclusivo

- **sorted()** : pode ser usada em qualquer iterável, incluindo tuplas.
  - números = (13, 2, 78, 3, 45, 67, 18, 7)
  - sorted(números) # [2, 3, 7, 13, 18, 45, 67, 78]
    - O *sorted()* a função sempre criará uma nova lista dos valores, ao contrário do *sort()* que não retorna uma nova lista.

- **reverse()** : para criar uma nova lista de valores na ordem inversa.
