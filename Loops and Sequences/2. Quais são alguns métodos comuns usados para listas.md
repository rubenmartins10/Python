- **append()** : este método é usado para adicionar um item ao final da lista. 
  - números = [1, 2, 3, 4, 5]
  - úmeros.append(6)
  - print(números) # [1, 2, 3, 4, 5, 6]
    - Também podemos utilizar esse método para colocar uma lista no fim de outra.

- **extend()** : este método dá-nos possibilidade de adicionar vários elementos de uma lista para outra.
  - números = [1, 2, 3, 4, 5]
  - números_par = [6, 8, 10]
  - números.extend(números_par)
  - print(números) # [1, 2, 3, 4, 5, 6, 8, 10]
      - Como podemos observar a lista aninhada desapareceu e é apenas uma lista de números

- **insert()** : para inserirmos um elemento num índice específico da lista. O método aceita 2 argumentos: o índice onde desejo inserir o novo item e o item que desejo inserir:
  - números = [1, 2, 3, 4, 5]
  - números.insert(2, 2.5)
  - print(números) # [1, 2, 2,5, 3, 4, 5]

- Se quiser remover algum elemento da lista utiliza-se o método **remove()**, e utilizamos o valor que queremos remover como argumento.

- **pop()** : para remover um elemento num índice específico da lista

- **clear()** : para esvaziar a lista.

- **sort()** : é usado para organizar os elementos por lugar:
  - números = [19, 2, 35, 1, 67, 41]
  - números.sort()
  - print(números) # [1, 2, 19, 35, 41, 67]

- **sorted()** : este método funciona para qualquer iterável e retorna uma nova lista organizada em vez de modificar a lista original. Este método é igual ao de cima, mas cria uma lista nova organizada.

- **reverse()** : reverte a lista de elementos existentes

- **index** : este método serve para encontrar o índice de um elemento da lista
  - programação_linguagens = ['Fuga', 'Java', 'Píton', 'C++']
  - programação_linguagens.index('Java') #1
