- Os **loops** são usados para repetir um bloco de código por um determinado número de vezes.

- **loop for** : 
  - programming_languages = ['Rust', 'Java', 'Python', 'C++']
  - for language in programming_languages:
    - print(language)
  - É de reparar que o print está dentro do loop, caso não estivesse receberíamos um **IdentationError**

- Podemos também fazer um loop para iterar através de outros iteráveis
  - for char em 'code'
    - print(char) 
      - O resultado é 'c' 'o' 'd' 'e', mas na vertical, uma letra em cada linha.

- Podemos fazer um loop dentro de um loop.

- Outro tipo de loop, é o **while**, que significa que ele irá repetir um bloco de código até que a condição seja falsa.

- O **break** é a instrução usada para interromper a execução de um loop
  - developer_names = ['Jess', 'Naomi', 'Tom']
  - for developer in developer_names:
    - if developer == 'Naomi':
        - break
    - print(developer)

- O **continue** é a instrução usada para pular a iteração atual de um loop e passar para a próxima iteração
  - developer_names = ['Jess', 'Naomi', 'Tom']
  - for developer in developer_names:
    - if developer == 'Naomi':
        - continue
    - print(developer)
