- numbers = [1, 2, 3, 4, 5]
- result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
- print(result)
  - Neste exemplo temos uma lista de números pares e queremos criar uma nova lista de tuplas indicando quais números são pares ou ímpares. Na primeira parte da compreensão da lista, usamos uma instrução **if** para verificar se o número é divisível uniformemente por 2. Se sim, então o resultado é uma tupla desse número seguida pela palavra **even**. Caso contrário, o resultado é uma tupla com o número seguido pela palavra **Odd**
  - O que aparece na consola:
    - [(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]

- Outra forma de criar uma lista a partir de um iterável existente é a função **filter()**. Um exemplo de criação de uma nova lista de apenas palavras com mais de 4 caracteres:
  - words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']
  - def is_long_word(word):
      - return len(word) > 4
  - long_words = list(filter(is_long_word, words))
  - print(long_words) # ['mountain', 'river', 'cloud']
  - **filter()** é a função usada para selecionar elementos de um iterável que atendam a uma condição específica. Aceita uma função e um iterável para os seus argumentos.

- Existem mais algumas funções que são úteis ao trabalhar com listas. Outra função é **map()**, que pega um iterável e aplica uma função a cada um dos seus elementos. Aqui está um exemplo de uso da função *map()* para converter uma lista de temperaturas de Celsius para Fehrenheit:
  - celsius = [0, 10, 20, 30, 40]
  - def to_fahrenheit(temp):
      - return (temp * 9/5) + 32
f-  ahrenheit = list(map(to_fahrenheit, celsius))
  - print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0] 

- Assim como *filter()*, *map()* aceita uma função e um iterável para os seus argumentos.

- Numa soma normal, basta apenas somarmos os valores e temos o resultado, utilizando o argumento **start** podemos definir um valor inicial para a soma
  - numbers = [5, 10, 15, 20]
  - total = sum(numbers, start=10) # keyword argument
  - print(total) # 60
