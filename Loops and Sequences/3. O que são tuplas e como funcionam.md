- Uma tupla é um tipo de dados usados para criar uma sequência ordenada de valores. as tuplas podem conter um conjunto misto de tipos de dados como este:
  - desenvolvedor = ('Alice', 34, 'Desenvolvedor de ferrugem')
  - As tuplas são semelhantes às listas, mas embora sejam um tipo de dados mutável, as tuplas são imutáveis. 
    - Os elementos de uma tupla não podem ser alterados depois de criada.
  - Se tentar atualizar um dos itens da tupla, obterei : **TypeError**

- Para acessar um elemento a partir de uma tupla, posso usar a notação de [ ] e o número do índice.

- Se precisar de acessar elementos a partir do final de uma tupla, usa-se a indexação negativa.

- Se tentar passar um número de índice que exceda ou seja igual ao comprimento da tupla, receberei um **IndexError**

- Outra maneira de criar uma tupla é a usar o construtor **tuple()**
  - Para o **tuple()** construtor, posso passar em diferentes iteráveis, como strings, listas, e até outras tuplas.
    - desenvolvedor = 'Jéssica'
    - tupla(desenvolvedor) # ('J', 'e', 's', 's', 'i', 'c', 'a')
  - Para verificar se há um certo item numa tupla, utilizamos a palavra-chave, **in**.
  - Também se pode desempacotar itens de uma tupla, da mesma maneira com listas.
  - Se precisar de coletar qualquer elemento restantes de uma tupla, posos usar o operador **'*'**
    - developer = ('Alice', 34, 'Rust Developer')
    - name, *rest = developer
    - print(name) # 'Alice'
    - print(rest) # [34, 'Rust Developer']
  - Assim como numa lista também, podemos usar o operador **' : '** para fazer uma extração de uma parte da lista.
    - desserts = ('cake', 'pie', 'cookies', 'ice cream')
    - desserts[1:3] # ('pie', 'cookies')

- Não se pode remover um item de uma tupla porque as tuplas são imutáveis.

- Se precisar de uma coleção dinâmica de elementos onde possa adicionar, remover e atualizar elementos, usamos uma **lista**. Se sei que estou a trabalhar com uma coleção fixa e imutável de dados, devo usar uma **tupla**.
