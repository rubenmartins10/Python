- **range()** : a função é usada para gerar uma sequência de inteiros.
  - range(start, stop, step)
    - O argumento *stop* é necessário e é um argumento que é um número inteiro que representa o ponto final para a sequência de números que está a ser gerado.
      - for num in range(3):
        - print(num)
- Se *start* agumento não é especificado, então o padrão é 0. Caso contrário, posso usar o *start* argumento para iniciar a sequência de inteiros num número inteiro diferente de 0.
- Aqui está um exemplo da geração entre 1 e 4:
  - for num in range(1, 5):
    - print(num)
- Por padrão, a sequência de inteiros aumentará 1. Mas se quiser alterar esse padrão, poderei usar o agumento opcional *step*. Aqui está um exemplo de geração de uma sequência de inteiros pares entre 2 e 10:
  - for num in range(2, 11, 2):
    - print(num)
- Como mencionado anteriormente, há apenas um argumento necessário para o **range()**. Se não fornecer nenhum argumento então recebo : **TypeError**
- Se eu desejar gerar uma sequênciad e inteiros em ordem decrescente, então:
  - for num in range(40, 0, -10)    
    - print(num)
- Posso utilizar a função range para criar uma lista de inteiros usando o construtor **list()**. O construtor **list()** é usado para converter um iterável numa lista
  - números = list(range(2, 11, 2))
  - print(números) #[2, 4, 6, 8, 10]
