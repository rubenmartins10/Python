- **enumerate()** : mantém o controlo do índice para um iterável e retorna um objeto numerado.

- Se passarmos a lista *languages* para o *enumerate*, converte o valor retornada numa lista com a função *list()*:
  - languages = ['Espanhol', 'Inglês', 'Russo', 'Chinês'] 
  - list(enumerate(languages)) # [(0, 'espanhol'), (1, 'inglês'), (2, 'russo'), (3, 'chinês')]

- languages = ['Espanhol', 'Inglês', 'Russo', 'Chinês']
- for index, languages in enumerate(languages):
