- O tipo de dados de uma lista é uma sequência ordenada de elementos que podem ser compostos de strings, números ou até mesmo outras listas.
  - As listas são mutáveis e usam indexação baseada em 0.

- Exemplo de uma lista:
  - cidades = ['Los Angeles', 'Londres', 'Tóquio']
  - Vamos nesta lista fazer referência ao elemento com índice 0
    - cidades[0] #'Los Angeles'

- Outra maneira de criar uma lista, é usar o construtor **list()**. É usado para converter um iterável numa lista como esta:
  - desenvolvedor = 'David'
  - list(desenvolvedor) # ['D', 'a', 'v', 'i', 'd']

- Para obter um número total de elementos numa lista, utilizamos o **len()**

- Podemos atualizar  o valor num índice específico da lista:
  - linguagens_programacao = ['Python', 'Java', 'C++']
  - linguagens_programacao[0] = 'C'
  - print(linguagens_programacao) #['C', 'Java', 'C++']

- Como as listas são mutáveis, posso atualizar qualquer elemento da lista, desde que passe por um número com índice válido. Se passar um índice positivo ou negativo, fora dos limites da lista, recebo: **IndexError**

- Se quiser remover um elemento de uma lista, uso o **del** seguido do nome da lista e o índice que queremos remover.

- Podemos também verificar se um elemento está dentro da lista, para isso escrevemos o nome do devido elemento que queremos confirmar que está na lista e utilizamos o **in**
  - 'Python' in linguagens_programacao #True

- É comum também ter listas aninhadas:
  - desenvolvedor = ['Alice', 25, ['Píton', 'Fuga', 'C++']]
  - desenvolvedor[2] #['Python', 'Rust', 'C++']

- **Desempacotar** valores de uma lista é uma técnica usada para atribuir valores de uma lista a novas variáveis.
  - desenvolvedor = ['Alice', 34, 'Desenvolvedor de ferrugem']
  - nome, idade, trabalho = desenvolvedor
    - Se precisar de recolher quaisquer elementos restantes de uma lista, poderei usar o **'*'** operador assim:
      - developer = ['Alice', 34, 'Rust Developer']
      - name, *rest = developer
      - print(name) # 'Alice'
      - print(rest) # [34, 'Rust Developer']
    - Se o número de variáveis do lado esquerdo do operador de atribuição não corresponder ao número total de itens na lista, aparece: **ValueError**

- Veremos também o operador **' : '**, semelhante às strings, pode-se acessar partes de uma lista usando o operador:
  - desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
  - desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
    - O índice inicial é **1** já que isso aponta para o segundo item da lista, e o índice final é **4**, que inclui tudo, mas não o inclui a ele mesmo.
  - Outra coisa que podemos fazer com esse operador é especificar um intervalo de passos que determina quanto incrementar entre os índices.
    - números = [1, 2, 3, 4, 5, 6]
    - números[1::2] # [2, 4, 6] //dá apenas uma lista de números pares
      - O primeiro número par está no índice **1** então esse será o índice inicial. Como queremos passar para o final da lista,, omitimos o índice final. Por último, especificamos 2 para o intervalo de passo opcional, ele só aumentará **2** em vez do padrão **1**.
