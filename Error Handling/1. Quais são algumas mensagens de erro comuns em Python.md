- Erros comuns em python incluem:
  - **SintaxeError**
    - Aparece quando o código não segue as regras de sintaxe adequadas.
  - **NameError**
    - Aparece quando não consegue encontrar uma variável com esse nome.
  - **TypeError**
    - Aparece quando tento realizar uma operação em tipos de dados incompatíveis.
  - **IndexError**
    - Aparece quando saio dos limites, normalmente de uma lista ou assim.
  - **AttributeError**
    - Aparece quando tento usar um método ou propriedade que não existe para esse tipo de dados.
