- A *depuração* é o processo de identificação e resolução de erros ou bugs no código

- Usando **print()** e **f-strings**
  - Quando utilizamos o print() função e f-strings em vários pontos do código:
    - def add(a, b):
      - result = a +b
      - print(f'Added {a} e {b} results in {result}')
      - return result
    - Substitui os valores a, b e result pelos valores esperados

- **Usando um debugger VS**
  - *1 Passo: Configurar o código*
    - Criar um ficheiro main.py
  - *2 Passo: Definir um ponto de interrupção*
    - 1. Clicar na margem esquerda ao lado de alguma linha e definir um ponto de interrupção
    - 2. Um ponto vermelho aparecerá, indicando que o ponto de interrupção está definido
  - *3 Passo: Iniciar a depuração*
    - 1. Carregue F5 ou em Executar -> Iniciar debugging
    - 2. Selecionar o ficheiro python
    - 3. O debug irá pausar a execução no ponto de interrupção
  - *4 Passo: Inspecionar as variáveis*
    - Passar o rato sobre as variáveis para ver os seus valores atuais
    - Usar o painel de variáveis para ver todas as variáveis locais
    - Usar a consola de debugging na parte inferior para avaliar expressões
  - *5 Passo: Passo através do código*
    - Continuar (F5): Retomar a execução até o próximo ponto  de interrupção
    - Passo acima (F10): Execute a linha atual e passe para a próxima
    - Entre (F11): Entrar em chamadas de função
    - Saia (Shift+F11): Saia da função atual
