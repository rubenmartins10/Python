- **raise** statement é uma ferramente que permite acionar manualmente exceções no código. Oferece controlo sobre quando e como os erros sãon gerados, permitindo criar condições de erro personalizadas e impor comportamento específico do programa
  - É usada para lançar explicitamente uma exceção em qualquer ponto do programa, permitindo sinalizar que ocorreu uma condição de erro ou que determinados requisitos não foram atendidos

- A variável **de** é a palavra-chave usada para encadear exceções
