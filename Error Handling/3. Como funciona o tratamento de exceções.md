- O tratamento de exceções é o processo de captura e gestão de erros que ocorrem durante a execução de um programa, para que o código não pare inesperadamente.
- **try**
  - O bloco de código onde prevejo que pode acontecer um erro
- **except**
  - É executado se um erro do tipo especificado for levantado dentro da tentativa
- **else**
  - Executa se nenhuma exceção for levantada no bloco try
- **finally**
  - Funciona independentemente do que acontecer, independentemente de ter ocorrido ou não uma exceção. Útil para tarefas de limpeza, como fechar arquivos
- **e**
  - Posso utilizar e para permitir o acesso à mensagem de erro ou objeto real para registo ou debugging
