- Para determinar corretamente o escopo, o Python segue o **LEGB** regra, que significa:
  - **Âmbito local (L)** : variáveis definidas em funções ou classes
  - **Escopo envolvente (E)** : variáveis definidas em funções enclosing ou aninhadas.
  - **Escopo global (G)** : variáveis definidas no nível superior do módulo ou ficheiro.
  - **Escopo integrado (B)** : nomes reservados em python para funções, módulos, palavras-chave e objetos predefinidos.

- *Âmbito local* significa que uma variável declarada dentro de uma função ou classe só pode ser acessada dentro dessa função ou classe.
  - def meu_func():
    - meu_var = 10
    - print(meu_var)
  - O *meu_func* a função tem o seu próprio escopo que não pode ser acessado fora da função. A variável sairá com valor 10, mas o print da variável fora da função vai levar a *NomeErro*

- *Escopo envolvente* significa que uma função aninhada dentro de outra função pode acessar as variáveis da função na qual está aninhada.
  - def outer_func():
    - msg = 'Ola ai'
    - def inner_func():
      - print(msg)
    - inner_func()
  - outer_func() #Ola ai
- A função interna *inner_func*, pode acessar livremente o *msg*, variável definida na função exterior, *outer_func*. Mas as funções externas não podem acessar variáveis definidas em nenhuma função aninhada.

- *Âmbito global* refere-se a variáveis declaradas fora de quaisquer funções ou classes que possam ser acessadas de qualquer lugar do programa. Nesta, meu_var pode ser acessado em qualquer lugar, mesmo dentro de uma função na qual não está definido
  - meu_var = 100
    - def mostrar_var():
      - print(meu_var)
  - mostrar_var() #100
  - print(meu_var) #100
- E se quiser tornar acessível globalmente uma variável com escopo local definida dentro de uma função, pode-se usar o global palavra-chave

- *Escopo integrado* refere-se a todas as funções, módulos e palavras-chave integrados do python e estão disponíveis em qualquer lugar do programa:
  - print(str(45)) # '45'
  - print(tipo(3.14)) # <classe 'float'>
  - print(isistance(3, str)) # Falso
