- Um **fatiamento de corda** permite extrair uma parte de uma string ou trabalhar apenas com uma parte específica dela. Sintaxe:
  - corda[começar:parar]
- Se quiser extrair caracteres de um determinado índice para o outro, basta separar o começar e parar índices com dois pontos:
  - meu_str = 'Olá mundo'
  - print(meu_str[1:4]) #la
    - Notar que o *parar* o índice não é inclusivo, portanto [1:4] acabei de extrair os caracteres do índice 1, e até, mas não incluindo, o caractere no índice 4

- Posso também omitir o começar e parar índices e esses serão padronizados 0 ou o fim da string, respetivamente. 
  - Exemplo se quiser omitir o começar:
    - meu_str = 'Olá mundo'
    - print(meu_str(:7)) #Olá mund
      - Isto extrai tudo do índice 0 até ao caractere no índice 7
  - Quando omitimos o parar é exatamente igual mas em vez de acabar num índice, começa nesse índice.

- Fatiar uma string não modifica a string original.

- Também poderei omitir ambos, começar e parar índices, que vão extrair a string toda sem nenhuma alteração

- Além do começar e parar índices, também há um opcional *passo* parâmetro, que é usado para especificar o incremento entre cada índice na fatia.
  - Sintaxe:
    - corda[começar:parar:passo]
  - No exemplo abaixo, o corte começa no índice 0, para antes 11, e extrai cada segundo caractere:
    - meu_str = 'Hello World'
    - print(meu_str[0:11:2]) #Hlowrd

- Posso reverter uma string definindo a etapa para -1, e omitindo começar e parar
  - meu_str = 'Hello world'
  - print(meu_str[::-1]) #dlrow olleH
