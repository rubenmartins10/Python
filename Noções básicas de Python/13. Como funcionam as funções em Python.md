- **int()** converte um número, booleano e uma string numérica num número inteiro.

- Usamos o **def** para definir funções personalizadas, e em seguida o nome que queremos dar à função. 
    - Para executar a função precisamos de a chamar seguida de uns parentes.

- As funções também usam um especial **return** palavra-chave para sair da função e retornar um valor, se não usarmos **return**, python vai retornar **None** por omissão.
