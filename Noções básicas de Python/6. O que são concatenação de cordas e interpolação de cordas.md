- Em Python podemos combinar várias strings utilizando o operador **+**. Este processo é chamado de **concatenação de string**.
  - meu_str_1 = 'Olá'
  - meu_str_2 = "Mundo"
  - str_plus_str = meu_str_1 + ' ' + meu_str_2
  - print(str_plus_str) # Olá Mundo
    - Este processo só funciona quando concatenamos duas strings, se tentarmos concatenar uma string com um número, obtrei um **TypeError**.
    - nome = 'John Doe'
    - idade = 26
    - nome_e_idade = nome + idade
    - print(nome_e_idade) # TypeError: só pode concatenar str (não "int") para str

- Isto acontece porque o Python não converte automaticamente outros tipos de dados, como inteiros, sem strings quando os tento concatenar.Python requer que todos os elementos sejam strings antes de os concatenar.
  - Para corrigir tal erro e converter um número numa string podemos usar uma função **str()** com o número dentro.

- Também posso usar o operador de atribuição aumentada para concatenação. Isso é representado pelo sinal **+=**, e realiza concatenação e atribuição numa única etapa.

- O processo de inserção de variáveis e expressões numa string é chamado de **interpolação string**. Python tem uma categoria de string chamada **cordas f**, que permite lidar com a interpolação com uma sintaxe compacta e legível.
  - As cordas F começam com **f** antes das aspas e permitem incoporar variáveis ou expressões dentro de campos de substituição indicados por chaves {}. 
    - Exemplo:
      - nome = 'John Doe'
      - idade = 26
      - nome_e_idade = f'Meu nome é {nome} e EU sou {idade} anos'
      - print(nome_e_idade) # Meu nome é John Doe e tenho 26 anos
      - num1 = 5
      - num2 = 10
      - print(f'A soma de {num1} e {num2} é {num1 + num2}') # A soma de 5 e 10 é 15
