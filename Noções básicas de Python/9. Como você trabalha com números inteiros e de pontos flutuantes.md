- Os números inteiros são números inteiros sem vírgulas decimais, positivas ou negativas

- Como realizar uma operação de adição com números inteiros:
  - my_int_1 = 56
  - my_int_2 = 12
  - sum_ints = my_int_1 + my_int_2
  - print('Integer Addition:', sum_ints) # Integer Addition: 68
    - É exatamente igual para subtração, multiplicação e divisão.

- Floats são números positivos ou negativos com vírgulas decimais, como: **3.13, -0.5 ou 0.0**.
  - Exatamente igual todas as operações como se fossem um inteiro.

- Se fizermos qualquer tipo de operação entre um número inteiro e um float, o resultado automaticamente será convertido automaticamente num float.

- O operador módulo **%** retorna o restante quando o valor à esquerda é dividido pelo valor à direita

- A divisão de piso divide dois números e retorna o maior número inteiro menor ou igual ao resultado. Isso é feito com o operador de barra dupla **//**

- A exponenciação eleva um núermo à potência de outro e é feita com o operador de asterisco duplo "**".

- O **float()** a função retorna um número de ponto flutuante construído a partir do número fornecido.

- O **int()** a função retorna um número inteiro construído a partir do número fornecido.

- **Round()** : arredonda um número para o número especificado de casas decimais. Por padrão, esta função arredonda para o número inteiro mais próximo e retorna um número inteiro sem casas decimais.

- **Abs()** : retorna o valor absoluto de um número.

- **Pow()** : eleva um número à potência de outro ou realiza exponenciação modular.
