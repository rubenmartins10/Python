- Python é uma linguagem de tipo dinâmico como JavaScript, o que significa que você não precisa declarar explicitamente tipos de variáveis. A linguagem sabe qual tipo de dados uma variável é baseada no que você atribui a ela.

- A codificação em Python é muito rápida e mais flexível, isso pode levar a bugs inesperados porque erros de tipo são detetados apenas quando o **programa é executado e não quando o programa é compilado.**
  - Python executa o código linha por linha, se atingir uma linha onde se espera que um determinado objeto se comporte de uma forma que não é capaz, o Python irá parar e mostrar um erro.

- No tipo Python, os erros podem se revelar durante a execução, quando o programa estiver realmente em execução e usando seu código.

- As linguagens compiladas detectam erros de tipo durante a etapa de compilação, antes que o programa possa ser executado.

my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 5, 8}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 5, 8)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>


- O embutido isinstância() a função permite verificar se uma variável corresponde a um tipo de dados específico. Ele recebe um objeto e o tipo em que você deseja verificá-lo e, em seguida, retorna um booleano.
  - Exemplo:

isinstance('Hello world', str) # True
isinstance(True, bool) # True
isinstance(42, int) # True
isinstance('John Doe', int) # False
