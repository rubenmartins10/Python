- Em python, cada valor tem um valor booleano inerente, ou uma sensação embutida ele deve ser tratado como *Verdadeiro* ou *Falso* num contexto lógico. Muitos valores são considerados **truthy**, ou seja, eles avaliam *Verdadeiro* num contexto lógico. Outros são **falsidade**, o que significa que eles avaliam *Falso*.

- Alguns valores de falsidade:
  - None
  - False
  - Inteiro 0
  - Float 0.0
  - Aspas vazias " "

- Outros valores, como números diferentes de zero e strings não vazias, são verdadeiros.

- Se quiser verificar se um valor é verdadeiro ou falso, posso usar a função **bool()**.

- Existem três operadores booleanos em python:
  - AND
  - OR
  - NOT

- O operador **AND** pega dois operandos e retorna o primeiro operando se for falsidade, caso contrário, ele retorna o segundo operando. Ambos os operandos devem ser truthy para que uma expressão resulte num valor verdadeiro

- O operador **OR** retorna o primeiro operando se for verdadeiro, caso contrário, retorna o segundo operando. Num **OR** a expressão resulta num valor verdadeiro se pelo menos um operando for verdadeiro.

- O operador **NOT** converte os valores falso em verdadeiro, e vice-versa.
