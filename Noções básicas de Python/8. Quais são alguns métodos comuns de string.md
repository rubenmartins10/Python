- **Upper()** : retorna uma nova string com todos os caracteres convertidos em maiúsculas

- **Lower()** : retorna uma nova string com todos os caracteres convertidos para minúsculas

- **Strip()** : retorna uma nova string com os caracteres iniciais e finais especificados removidos. Se nenhum argumento for aprovado, ele removerá os espaços em branco iniciais e finais

- **Replace(old,new)** : retorna uma nova string com todas as ocorrências de velho substituído por novo

- **Split(separator)** : divide uma string num separador especificado numa lista de strings. Se nennhum separador for especificado, ele divide-se em espaços em branco.

- **Join(iterable)** : une os elementos de um iterável numa string com um separador

- **Startswith(prefix)** : retorna um booleano indicando se uma string começa com o prefixo especificado

- **Endswith(suffix)** : retorna um booleano indicando se uma string termina com o sufixo especificado

- **Find(substring)** : retorna o índice da primeira ocorrência de substring, ou -1 se não encontrar uma

- **Count(substring)** : retorna o número de vezes que uma substring aparece numa string

- **Capitalize()** : retorna uma nova string com o primeiro caractere maiúsculo e os demais caracteres minúsculos

- **Isupper()** : devolve *Verdadeiro* se todas as letras da string forem maiúsulas e *Falso* se não

- **Islower()** : devolve *Verdadeiro* se todas as letras da string forem minúsculas e *Falso* se não

- **Title()** : retorna uma nova string com a primeira letra de cada palavra maiúscula
