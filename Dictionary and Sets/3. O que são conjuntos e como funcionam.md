- Os conjuntos são uma das estruturas de dados integradas no python. Uma das principais características dos conjuntos é que eles não armazenam valores duplicados.
  - Se tentar adicionar um valor duplicado a um conjunto, apenas um deles será armazenado.
- Os conjuntos são mutáveis e não ordenados, o que significa que os seus elementos não são armazenados em nenhuma ordem específica, portanto, não posso usar índices ou chaves para acessá-los.
  - Só podem conter valores de tipos de dados imutáveis, como números, strings e tuplas
  - my_set = {1, 2, 3, 4, 5}
  - Uma peculiaridade de trabalhar com conjuntos é que se precisamos de definir um conjunto vazio, deverei usar o **set()**, e **{ }** ele automaticamente cria um dicionário.
- Para adicionar algum elemento a um conjunto, utilizamos o **.add()**
  - my_set.add(6)
- Se eventualmente tentarmos adicionar um número que já está na lista, ele não duplica valores, logo  alista não vai mudar.

- Temos duas opções pare remover um elemento do conjunto:
  - **.remove()**
  - **.discard()**
    - O método *.remove()* levará a erro **KeyError** se o elemento não for encontrado, enquanto o *.discard()* não
  - **.clear()** : remove todos os elementos do conjunto

- **.issubset() e .issuperset()** são métodos que verificam se um conjunto é um subconjunto ou superconjunto de outro conjunto. Retorna True ou False
  - my_set = {1, 2, 3, 4, 5}
  - your_set = {2, 3, 4, 6}
  - print(your_set.issubset(my_set)) #False
  - print(my_set.issuperset(your_set)) #False

- **.isdisjoint()** é o método que verifica se dois conjuntos são disjuntos, o que significa que eles não têm nenhum elemento em comum.
  - print(my_set.isdisjoint(your_set)) #False

- O operador **|** indica o retorno de um novo conjunto com todos os elementos de ambos os conjuntos
  - my_set | your_set # {1, 2, 3, 4, 5, 6}

- O operador **&** indica o retorno de um novo conjunto apenas com os elementos que os conjuntos têm em comum
  - my_set & your_set # {2, 3, 4}

- O operador **-** retorna um novo conjunto com os elementos do primeiro conjunto que não estão nos outros conjuntos
  - my_set - your_set # {1, 5}

- O operador **^** retorna um novo conjunto com os elementos que estão no primeiro ou no segundo conjunto, mas não em ambos
  - my_set ^ your_set #{1, 5, 6}
