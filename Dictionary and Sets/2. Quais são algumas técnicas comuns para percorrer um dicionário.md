- product = {
  - 'Laptop' : 990,
  - 'Smartphone' : 600,
  - 'Tablet' : 240,
  - 'Headphones' : 70
- }
- **.values(), .keys() e .items()** são métodos essenciais para essa técnicas. 
  - Retornam um objeto de visualização com os valores, chaves e pares de valores-chave do dicionário. Posso usar esses objetos de visualização em **for** loops para iterar os elementos.

- Escrevo um **for**, a variável de loop (price) neste caso para obter todos os valores do **product** no dicionário
  - for price in product.values():
    - print(price)

- Se precisar de iterar os pares chave-valor enquanto acompanho um contador, posso chamar a função **enumerate()**. Este contador atua essencialmente como uma espécie de índice para o elemento dentro do loop
  - A função retorna um **enumerate** objeto, que atribui um número inteiro a cada par chave-valor, como um contador. Posso iniciar o contador a partir de qualquer número, mas por padrão, ele começa a partir de 0
  - for product in enumerate(product):
    - print(product)
      - Saída:
      - #(0, 'Laptop')
      - #(1, 'Smartphone')
      - #(2, 'Tablet')
      - #(3, 'Fones de ouvido')
  - Dá-nos tudo o que pedirmos de forma enumerada começando por 0.
