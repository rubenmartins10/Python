- Em python os dicionários são estruturas de dados integradas que armazenam coleções de pares chave-valor.
  - Com dicionários python, uso uma chave para encontrar o  valor correspondente. Devo usar dicionários quando preciso de associar valores a chaves exclusivas.
    - Útil quando preciso de encontrar um valor rapidamente com base na chave e quando preciso de representar dados estruturados.
    - dicionario = {
      - chave1: valor1,
      - chave2: valor2
    - }

- Cada chave está associada a um valor, para que possa usar a chave para acessar esse mesmo valor. Após cada valor, exceto o último, há uma vírgula para separar os diferentes pares chave-valor.
  - As chaves devem ser únicas no dicionário e devem de ser um tipo de dados imutável.

- pizza = {
    - 'name': 'Margherita Pizza',
    - 'price': 8.9,
    - 'calories_per_slice': 250,
    - 'toppings': ['mozzarella', 'basil']
-}
  - O dicionário é atribuído a *pizza* como variável. Possui quatro pares chaves-valor: *name, price, calories_per_slice e toppings*
  - Outra alternativa seria usar o **dict()**, como construtor a partir de uma sequência de pares de chaves-valor
    - pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
  - Se eu quisesse acessar o valor nome da pizza, escreveria assim:
    - pizza['name']
  - Se a chave não existir no dicionário, um novo par chave-valor será criado. Os dicionários preservam a ordem de inserção, isso é útil quando preciso iterar pelo dicionário:
    - pizza['name'] = 'Margherita'
      - E agora o valor da chave *name* é *'Margherita'*

- **.get()** : recupera o valor associado a uma chave. A vantagem é que posso definir um valor padrão, para não obter um erro se a chave não existir:
  - dict.get(key, default)

- **.items()** : o método retorna um objeto de visualização com todos os pares chave-valor no dicionário, incluindo as chaves e os valores

- **.clear()** : método que remove todos os pares chave-valor do dicionário

**.pop()** : o método remove o par chave-valor com a chave que especifico como primeiro argumento e retorna o seu valor. Se a chave não existir, ela retorna o valor padrão que se especifica como segundo argumento. Se a chave não existir e não passar um valor padrão, dá-nos então um **KeyError**

- **.popitem()** : método que remove o último item inserido

- **.update()** : o método atualiza os pares chave-valor com os pares chave-valor de outro dicionário. Se eles tiverem chaves em comum, os seus valores serão substituídos.
