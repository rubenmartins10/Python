- Em vez de ter que implementar cada parte do código do zero, uma biblioteca fornece código pré-escrito e reutilizável, como funções, classes e estruturas de dados.
- Python possui uma extensa biblioteca padrão com muitos módulos integrados diferentes. São todas as soluções padronizadas e bem avaliadas para muitos dos problemas e tarefas que se enfrentará diariamente:
  - Interagindo com o sistema operacional
  - Trabalhando com arquivos
  - Rede
  - Trabalhando com data e hora
  - Realizando operações matemáticas
  - Usando expressões regulares
  - Testando e depurando o código

- **math**, o módulos possui funções úteis para realizar operações matemáticas mais complexas.

- **random**, o módulo é útil para gerar números aleatórios

- **re**, o módulo é usado para trabalhar com expressões regulares

- **datatime**, o módulo é útil para trabalhar com datas e horários

- Para importar qualquer tipo de biblioteca o que fazemos é 
  - import module_name

- Se precisar de chamar um método desse módulo no meu script, tenho de fazer:
  - module_name.method_name()

- Se precisar de importar o módulo com um nome diferente, posso usar essa sintaxe, mas com um **as**
  - import module_name as modulo_alias

- Por exemplo, posso me referir a **math** módulo com **m**, e depois posso chama-lo dessa maneira:
  - import math as m
  - m.sqrt(36) //raiz quadrada de 36

- Agora a função de import começa com **from**, seguido do nome do módulo e depois do **import** palavra-chave seguida do nome dos elementos que desejo importar:
  - from module_name import name1, name2
- Se quiser atribuir *alias* a esses nomes, posso fazer isso utilizando *as* :
  - from module_name import name1 as alias1, name2 as alias2

- Digamos que apenas queiramos importar radianos, seno e cosseno do módulo de matemática, e não toda a biblioteca:
  - from math import radians, sin, cos

  - __name__ é uma variável integrada especial em python
    - Quando um arquivo é executado diretamente, o python define o valor desta variável para a string '__main__'
    - Mas se o arquivo python for importado como módulo para outro script python, o valor do __name__ a variável é definida como o nome desse módulo (geralmente o nome do arquivo sem o **.py** extensão)
    - É por isso que vou encontrar esse condicional em scripts python, contém o código que desejo executar **só se** o script python está a ser executado como o programa principal
- Os scripts python têm 2 propósitos:
  - Podem ser executados diretamente para executar a sua lógica principal
  - Podem ser importados para outro módulo sem executar a sua lógica principal.
