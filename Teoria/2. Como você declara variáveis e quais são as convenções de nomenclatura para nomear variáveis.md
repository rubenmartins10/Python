- Atribuímos um valor a um identificador utilizado =   
  - Exemplo:
    - nome = 'Rúben Martins'
    - idade = 25
  - A variável nome detém o valor 'Rúben Martins', este texto é uma string, por isso tem de estar sempre entre aspas.

- Não podemos inicializar variáveis :
  - Os nomes das variáveis só podem começar com uma letra ou um sublinhado.
  - Os nomes das variáveis só podem conter caracteres alfanuméricos.
  - Os nomes das variáveis são sensíveis a maiúsculas e minúsculas.
  - Nomes de variáveis não podem ser uma das palavras-chave reservadas do Python, como:
    - If
    - Class
    - Def
