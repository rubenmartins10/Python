- Uma string é uma sequeência de caracteres cercados por aspas simples ou duplas.

- Na maioria das linguagens de programação, os caracteres rodeados por aspas são tratados de forma diferente dos caracteres rodeados por apas duplas, mas em Python são tratados de forma igual.

- Para fugir das aspas simples ou duplas na string, podemos usar uma barra invertida \.
  - Com este método posso usar aspas simples ou duplas para envolver a própria string:
    - msg = 'É um dia ensolarado'
    - citar = "Ela disse: \"Olá!\""

- Às vezes, pode ser necessário verificar se uma string contém um ou mais caracteres.
  - Para isso Python fornece o operador **in**, que retorna um booleano que especifica se o caractere ou caracteres existem na string ou não
    - meu_str = 'Olá mundo'
    - print('Olá' em meu_str)  # Verdadeiro

- Podemos obter o comprimento de uma string utilizando a função **len()**. 
  - Exemplo:
    - meu_str = "Olá Mundo"
    - print(len(meu_str)) ##11

- Cada caractere numa string possui uma posição chamada **índice**. O índice começa em 0, o que significa que o primeiro caractere de uma string é 0, e o segundo é 1, e assim em diante.
  - Para acessar a um caractere pelo seu indíce, usamos [] com o índice de caractere que desejo acessar dentro.
  - A indexação negativa também é permitida, para que possa obter o último caractere de cada string -1 e o penultimo caractere com -2, e assim em diante.
